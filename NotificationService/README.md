# Notification Service
Microservice that receives kafka messages and transpose it to firebase push notifications

![build](https://img.shields.io/github/actions/workflow/status/iNikitaGricenko/ComputerSystem/build.yml?style=for-the-badge/badge.svg)

# Technologies

## Architecture
* Microservice

## Backend
* Spring Boot
* Spring Cloud
* Apache Kafka
* Google Firebase
* Swagger
* JPA
* AspectJ
* Aspect oriented programming

### Code Write
* Lombok
* Mapstruct

### Services
* Github actions
* Docker & Docker-compose
