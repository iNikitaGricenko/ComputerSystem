# Email service
Microservice that sends emails to users/clients

# Technologies

## Architecture
* Microservices

## Backend
* Spring Boot
* Spring Cloud
* Spring Webflux
* Spring Validation
* Spring Mail
* Swagger
* Eureka Client
* Thymeleaf
* Spring Boot Actuator
* Openfeign

#### Mail Templates
* Template Engine - Thymeleaf

### Code Write
* Lombok
* Mapstruct

### Services
* Github actions
* Docker & Docker-compose
